# Tetris

Implemented in odin language with raylib

Work on platform which odin support

- Windows
- MacOS
- Linux
- ...

## Build

odin build . -o:size

On Windows add another flag to disable console

odin build . -o:size -subsystem:windows

## Dev

odin run .
